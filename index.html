    Automatic Variational Inference for Gaussian Process models.

    Code for the paper "Automated Variational Inference for Gaussian Process Models". If you use any part of the code, please cite: Nguyen, T.V., and Bonilla, E.V., Automated Variational Inference for Gaussian Process Models, In NIPS 2014

    Please see src/demoFull.m and src/demoMixture.m for how to use the code to perform exact inference for the regression model. Other models such as binary classification, multi-class classification, warped Gaussian processes, and log Gaussian Cox process, please see in the src/likelihood directory. Predictions for these models are also implemented in the **src/prediction" directory.
